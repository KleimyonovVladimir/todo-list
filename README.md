### Getting Started

1. _Clone the repo:_<br>
   `$ git clone https://github.com/KleimyonovVladimir/todo-list.git`

2. _Install dependencies:_ `yarn` or `npm install`

3. _Start the dev server:_ `yarn start` or `npm start`
